CMS Baukasten-System - Testseite

Übungsumgebung für das Baukasten-CMS

Nur zum Üben - Nicht produktiv!

Was können Sie hier tun?

  • Elemente hinzufügen: Fügen Sie neue Elemente zum elements-Array hinzu
  • Elemente entfernen: Löschen Sie Elemente aus dem Array
  • Elemente verschieben: Ändern Sie die Position der Elemente im Array
  • Inhalte ändern: Bearbeiten Sie die data-Objekte der Elemente
  • Neue Element-Typen erstellen: Fügen Sie neue Typen zu elementTypes hinzu

So bearbeiten Sie diese Seite:

  1. Öffnen Sie die Datei cms-test-config.js im Editor
  2. Ändern Sie die Elemente im pages.test.elements-Array
  3. Speichern Sie die Datei
  4. Laden Sie diese Seite neu (F5)
  5. Die Änderungen sind sofort sichtbar!
